# Calculadora de IMC em Python

Projeto desenvolvido em Python para calcular o Índice de Massa Corporal (IMC) de uma pessoa a partir do peso e altura informados.

## Sobre o projeto

A aplicação recebe os dados do usuário, realiza o cálculo do IMC e informa a classificação correspondente conforme o resultado obtido.

## Tecnologias utilizadas

- Python 3

## Funcionalidades

- Entrada de nome, peso e altura do usuário.
- Cálculo automático do IMC
- Classificação do resultado
